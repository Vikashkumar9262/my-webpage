# **App Name**: Alpha Terminal

## Core Features:

- Live Ticker Tape: Display a live ticker tape showing the current price of Nifty 50 and Bank Nifty, fetched from an external API.
- Stock Prediction Visualization: Embed interactive Plotly graphs to visualize stock prediction models versus actual historical data.
- Skills Radar Chart: Use a radar chart to visualize the balance between coding, finance, and math skills.
- Project Showcase: Display projects in a grid layout, including tech stack icons, key metrics, and links to source code and live demos.
- Resume and GitHub Links: Prominently display links to the resume and GitHub profile for easy access.
- Freelance Portfolio Highlight: Showcase freelance work, specifically mentioning the management of a portfolio using trend-following models since 2022.

## Style Guidelines:

- Color scheme: Dark mode with deep blues and blacks for a professional terminal look. Primary color: Deep navy blue (#24292F).
- Background color: Near-black (#121212), subtly lighter than the primary color.
- Accent color: Neon green (#39FF14) and red (#FF2400) for up/down indicators, call-to-action buttons, and highlights.  Green should indicate increases and red for decreases to give the effect of a professional trading dashboard
- Font: 'Source Code Pro' (monospace) for a code-like terminal feel, suitable for all text.
- Use minimalist icons representing technologies and skills. Icons should be in neon colors.
- Terminal-style layout with clear sections, grid layouts for projects, and well-defined spacing.
- Subtle animations for the ticker tape and graph updates.
